短信服务

框架

基于laravel 5.5+

环境要求

  • Composer
  • PHP >= 7.0.0
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

特别注意

凌凯用到的短信签名一定要跟我报备下,否则凌凯短信会被我拦截,从而发不出去!!!!

配置

修改.env 文件， 主要参数如下：

crontab

path/php artisan sms:send

path 为service服务根目录的完整路径，eg:xxx/sms-service/
