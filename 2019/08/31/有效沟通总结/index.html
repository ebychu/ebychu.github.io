如何有效沟通

发表于 2019-08-31 更新于 2019-09-02 分类于 Language Expression

起因

我不太善于表达自己想法，也不愿意跟人社交，有点儿小自闭。以前没觉得这样有什么不好，直到工作上无法避免地每天跟同事、上司沟通。因为自己不能有条理地、清晰地表达自己的想法，时常被否认被误解。

直到今天，压抑过久的情绪终于爆发出来，狠狠地在家哭了一场，哭过之后开始复盘自己的问题所在，以及该如何解决。最终定位在我的表达完全是以自我为中心的，从未站在别人的角度上思考过。

所以这次我想尝试换个角度，试想别人跟我沟通时，我想知道什么，才有了以下总结。

聆听者想要知道的信息

以下的每项内容可以根据实际场景调换顺序

  • 你有什么问题？你想问我什么？你想做什么
  • 为什么这么问？为什么这么做？为什么会有这个想法？
  • 希望先听到的内容是自己已知的，希望别人是站在理解自己的角度上问的
  • 他有什么独特的理由？

聆听者想要的感觉

  • 安全感
  • 时刻清晰当前在聊什么
  • 说话的人知道我的想法

安全感

安全感，对我来说，太抽象了，我无法感同身受别人的安全感。是以列出了我没有安全感时的情况，以此来推测别人的安全感是怎样的。

  • 生理领域被侵犯了（如：被挤痘痘）
  • 即将发生的事情不可预知（如：计划偏离）
  • 身体失控 （如：麻醉手术）
  • 认为目前的生理安全不受掌控 （如：恐高）
  • 可能面对名誉受损、财产受损、人际关系受损、地位权利受损
